## Install

MacOS

{% highlight text %}
brew install rust
{% endhighlight %}

## Initialize Project

{% highlight text %}
mkdir my_project
cd my_project
cargo init
{% endhighlight %}

## Build

{% highlight text %}
cargo build
{% endhighlight %}

## Run

{% highlight text %}
cargo run
{% endhighlight %}
